# 温州斜条批发 - UI 重设计规格

## 设计目标

在保持现有功能不变的前提下，将小程序 UI 从"传统批发市场红"升级为**简洁商务风**，提升美观度和易用性。

## 使用场景

- 主场景：B2B 客户自助浏览商品、下单、管理订单
- 次场景：老板/销售面对面给客户展示商品辅助下单

| 分割线 | `#EEEEEE` | 细线分隔 |

## 各页面设计

### 首页 (pages/index/index)

**顶部**：搜索栏 + 品类筛选
- 搜索栏：白底圆角输入框，左侧搜索图标，右侧清除按钮
- 分类标签：横向滚动，选中态主色白字，未选中灰底灰字

**产品双列网格**：
- 白底卡片，圆角 8rpx，阴影 `0 2rpx 12rpx rgba(0,0,0,0.06)`
- 内容：品类标签(左上) + 库存圆点(右上，绿/橙/红) + 产品名 + 占位图区(浅灰底) + 单价(主色加粗) + [+ 加入]按钮
- 缺货商品 [+] 按钮置灰不可点击

**底部悬浮购物车条**：
- 常驻底部，显示"共 N 件 | ¥XX.XX" + "去结算"主色按钮
- 点击展开完整购物车清单
- 换购模式下视觉区分（背景色提示）

### 购物车 (pages/cart/cart)

- 商品列表：白卡，每行勾选框 + 商品信息 + 数量调整 +/- + 删除
- 底部：全选 + 合计金额 + 去结算按钮

### 结算页 (pages/checkout/checkout)

从上到下：
1. **地址选择区**：白卡，显示收货人/电话/地址，右侧箭号切换地址簿；无地址时显示占位引导
2. **客户信息**：姓名/公司、手机号、详细地址三个输入框，左侧图标；手机号输入 ≥11 位自动匹配客户折扣，匹配成功显示绿色标签
3. **配送方式**：送货上门 / 到店自取 / 物流发货，三选一切换
4. **商品清单**：白卡，**每行一个商品**，格式：[商品名] ¥单价 [- 数量 +] = ¥小计；底部"添加商品"链接 + 合计行（有折扣时显示划线原价）
5. **备注 + 提交**：备注输入框(选填) + 勾选"保存地址到地址簿" + 底部主色"提交订单"大按钮

### 我的订单 (pages/orders/orders)

- 顶部状态筛选：全部 / 处理中 / 已完成 / 已取消
- 订单卡片：订单号+时间+状态标签 → 客户信息 → 商品概要 → 配送方式 → 金额 + 操作按钮区
- 操作按钮：取消订单、修改订单、再来一单、退换货
- 退换货弹窗：底部半屏，退货/换购切换，勾选商品+数量，换购跳首页选替代品，自动算差价
- 有退换申请时卡片显示退换状态标签

### 地址管理 (pages/address/address)

- 地址列表卡片，默认地址标记
- 新增/编辑表单，支持地图选点

### 后台登录 (pages/admin/login/login)

- 居中白卡，标题 + 密码输入 + 登录按钮，简洁克制

### 订单管理 (pages/admin/orders/orders)

- 顶部筛选：状态 + 搜索
- 订单卡片样式与客户端一致，额外操作：改状态、改价格、拍照上传、标记取货

### 产品管理 (pages/admin/products/products)

- 双列网格（与首页一致给管理员预览效果）
- 每张卡右上角编辑按钮
- 底部悬浮"+"按钮添加新产品

### 仪表盘 (pages/admin/dashboard/dashboard)

- 日期切换：今天/近7天/近30天
- 四个指标卡：总订单数、总销售额、待处理、未收款
- 柱状图：近7天每日销售额
- 排行：产品销量 Top10、客户消费 Top10
- 缺货预警列表

### 客户管理 (pages/admin/customers/customers)

- 搜索客户 + 列表卡片 + 折扣设置

## 设计 Token 体系

### 间距（8pt 栅格）

| Token | 值 | 用途 |
|-------|-----|------|
| space-xs | 4rpx | 图标与文字间距 |
| space-sm | 8rpx | 标签内部间距 |
| space-md | 16rpx | 卡片内边距 |
| space-lg | 24rpx | 板块间距 |
| space-xl | 32rpx | 页面大板块分隔 |
| space-2xl | 48rpx | 页面顶部/底部留白 |

### 字体层级

| Token | 大小 | 字重 | 用途 |
|-------|------|------|------|
| text-xs | 24rpx (12px) | 400 | 时间、标签、辅助文字 |
| text-sm | 28rpx (14px) | 400 | 描述文字、输入框 |
| text-base | 32rpx (16px) | 400 | 卡片标题、产品名 |
| text-lg | 36rpx (18px) | 600 | 页面标题 |
| text-price | 32rpx (16px) | 700 | 单价、合计金额 |
| 行高 | 1.5 | - | 全局正文行高 |

### 触摸交互规范

- 可点击元素最小尺寸：**88rpx × 88rpx**（44pt，iOS HIG / Material）
- 相邻点击目标间距：≥ 16rpx（8pt）
- 按钮按下反馈：opacity 降至 0.7，100ms 内响应
- 数量加减按钮扩展点击热区（图标小但命中区大）

### 动效 Token

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮点击反馈 | 150ms | ease-out |
| 卡片入场 | 200ms | ease-out |
| 弹窗进出 | 250ms | ease-in-out |
| 页面切换 | 300ms | ease-out |

### 阴影层级

| Token | 值 | 用途 |
|-------|-----|------|
| shadow-card | `0 2rpx 12rpx rgba(0,0,0,0.06)` | 产品卡、订单卡 |
| shadow-float | `0 4rpx 20rpx rgba(0,0,0,0.1)` | 悬浮购物车、FAB |
| shadow-modal | `0 -4rpx 24rpx rgba(0,0,0,0.12)` | 底部弹窗 |

## 实现要点

- `app.wxss` 定义 CSS 变量：配色、字号、间距、阴影、动效
- 卡片统一样式：白底、圆角 8rpx、shadow-card
- 状态色标：绿 `#27AE60` 充足/已完成、橙 `#E67E22` 偏低/待处理、红 `#C0392B` 缺货/已取消
- 所有可点击元素 ≥ 88rpx 热区
- 现有客户/后台功能逻辑不动，只换样式层
